# Ezra Kandie — Portfolio

Personal portfolio website for **Ezra Kandie**, a Full-Stack Developer and Data/AI Engineer based in Nairobi, Kenya.

🔗 **Live site:** https://ezrakandie.github.io/portfolio

---

## About

A fully hand-crafted, single-file portfolio with no frameworks, no build tools, and no dependencies beyond Google Fonts. Designed to load fast, look sharp, and represent real engineering work — not a template.

---

## Stack

- Pure HTML5, CSS3, and vanilla JavaScript
- Google Fonts: Space Grotesk · Inter · JetBrains Mono
- Zero npm packages · Zero build step · Deployable as a single `index.html`

---

## Features

- Custom cursor with lagging ring and glow follower effect
- Cycling hero title with smooth fade transitions
- Scroll-reveal animations on all sections
- Interactive project cards with live data visuals (charts, API inspectors, session monitors)
- Fully responsive — mobile nav included
- Dark theme with electric indigo accent

---

## Sections

Hero · About · Tech Stack · Projects · Experience · Education · Certifications · Contact

---

## Deploying

This is a single `index.html` file. To host on GitHub Pages:

1. Push `index.html` to the root of your repo
2. Go to **Settings → Pages → Source → main / root**
3. Save — live in ~60 seconds
